Animal Shelter (Backend) API

Welcome to the Animal Shelter API! You can use this API to view, create, update, and delete data from the shelter db.

Module Endpoint Method Description
Auth /auth/login POST Login with username and password
/auth/register POST Register a new user with username, password, email, and name
/auth/logout GET Logout manually (deleting the token is enough)
User /user GET List all users (id + username), admin access only
/user/me GET Display logged-in user's data
/user/:id GET Display specific user's data, self or admin access
/user POST Create a new user, admin can assign Role
/user/:id PUT Update specific user's data, self or admin access
/user/:id DELETE Delete a specific user, self or admin access
Media /media POST Upload file(s), field name: file
/media/* GET Access files in the public folder (e.g., /media/uploads/image.jpg)
Breed /breed GET Get all breeds
/breed/:id GET Get a breed by id
/breed POST Create a new breed
/breed/:id PUT Update a breed by id
/breed/:id DELETE Delete a breed by id (Sets breedId to null in all pets)
Species /species GET Get all species
/species/:id GET Get a species by id
/species POST Create a new species
/species/:id PUT Update a species by id
/species/:id DELETE Delete a species by id (Sets speciesId to null in all breeds)
Pet /pet GET Get all pets
/pet/:id GET Get a pet by id
/pet POST Create a new pet
/pet/:id PUT Update a pet by id
/pet/:id DELETE Delete a pet by id
/pet/status/:id GET Get the status history of a pet by id
