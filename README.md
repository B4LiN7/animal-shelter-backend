# Animal Shelter Backend
NestJS backend az Állatmenhely projekt számára.

## Futtatás
Feltétezzük, hogy a [Node](https://nodejs.org/)ot és a [Docker](https://www.docker.com/)t már telepítetted.
```
$ docker compose up --build
```
